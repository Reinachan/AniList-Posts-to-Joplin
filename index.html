AniList to Joplin Backup Tool

Easy-to-use for non-programmers

Note that Joplin seems to have a bug which means that it occasionally won't the note ID that I create through a function. That causes a duplication of those posts. If you get duplicate posts, delete all posts and run the tool again. This is likely a problem with the web-syncing tool

You can find it on a computer under settings > Web Clipper. Make sure to enable the webclipper as well. The port has to be "41184" and if it isn't, you'll have to edit the javascript.

You'll find this at the top of the same menu you found the access token in. It should be 41184, but if not, you'll have to type in that number in the box above

You can find this by going to your profile, opening your profile picture in a new tab and looking at the URL (page link). The user ID should be the numbers where I put X's in my example here: user/avatar/large/bXXXXXX-sga2KA4fswV1.png
